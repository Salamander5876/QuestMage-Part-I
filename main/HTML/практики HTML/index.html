Задание 1: Изображение с доступностью

Создай тег изображения с alt и добавь ленивую загрузку.

Подсказка: Используй loading="lazy" для экономии магии браузера.

Решение 
<img src="wand.jpg" alt="Волшебная палочка в руках мага" loading="lazy">

Задание 2: Метаданные для поисковых големов

Добавь метатеги description и keywords для страницы Академии.

Подсказка: keywords помогает големам понять тематику.

Решение 
<meta name="description" content="Академия Пустоты учит магии HTML">
<meta name="keywords" content="HTML, магия, код">

Задание 3: Безопасная ссылка

Создай ссылку на "Книгу заклинаний", открывающуюся в новой вкладке с защитой.

Подсказка: Добавь rel="noopener noreferrer" против тёмной магии хакеров.

Решение 
<a href="https://spells.com" target="_blank" rel="noopener noreferrer">Книга заклинаний</a>

Задание 4: Семантический каркас

Собери страницу с <header>, <nav>, <main> и <footer>.

Подсказка: <nav> — для меню, <main> — только один на страницу.

Решение 
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>Академия</title>
</head>
<body>
  <header>
    <h1>Академия Пустоты</h1>
  </header>
  <nav>
    <a href="#home">Домой</a>
  </nav>
  <main>
    <p>Добро пожаловать!</p>
  </main>
  <footer>
    <p>© 2025</p>
  </footer>
</body>
</html>

Задание 5: Форма для записи

Создай форму с полем имени, email и кнопкой. Добавь required.

Подсказка: Используй <label> для связи текста с полем через атрибут for.

Решение 
<form action="/signup" method="post">
  <label for="name">Имя:</label>
  <input type="text" id="name" name="name" required>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>
  <button type="submit">Записаться</button>
</form>

Задание 6: Спойлер с тайной

Создай элемент <details> с заголовком "Секрет" и скрытым содержимым.

Подсказка: Элемент <summary> отображает заголовок, а содержимое остается скрытым до клика.

Решение 
<details>
  <summary>Секрет</summary>
  <p>Магия HTML в чистоте кода!</p>
</details>

Задание 7: Видео-портал

Добавь видео через <iframe> с атрибутами для адаптивности.

Подсказка: Используй allowfullscreen для полноэкранного режима.

Решение 
<iframe width="560" height="315" src="https://www.youtube.com/embed/dQw4w9WgXcQ" title="Магия" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

Задание 8: Адаптивная картинка

Создай элемент <picture> с двумя версиями изображения для разных условий просмотра.

Подсказка: Используй элемент <source> для указания альтернативных версий изображения.

Решение 
<picture>
  <source media="(max-width: 600px)" srcset="small-spell.jpg">
  <img src="big-spell.jpg" alt="Заклинание">
</picture>
