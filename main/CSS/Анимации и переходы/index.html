1. Переходы: Плавное изменение состояния

Что такое переходы (transition)?

Переходы позволяют плавно изменять свойства элементов при наведении или других событиях. С помощью переходов можно задать длительность, задержку и функцию распределения для изменения таких свойств, как цвет, размер, положение и т.д.

Основные параметры:

  • transition-property: свойства, к которым применяется эффект;
  • transition-duration: длительность перехода;
  • transition-timing-function: функция распределения (например, ease, linear, ease-in-out);
  • transition-delay: задержка перед началом анимации.

Пример: Кнопка, которая плавно меняет цвет при наведении.

button {
  background-color: #7826e3;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
  transition: background-color 0.5s ease;
}

button:hover {
  background-color: #f8c291;
}

2. Анимации: Оживляем свитки с помощью @keyframes

Что такое анимации?

Анимации в CSS позволяют создавать последовательности изменений стилей, которые могут запускаться автоматически или по событию. Правило @keyframes определяет промежуточные этапы анимации, а затем их можно применить с помощью свойств animation.

Основные свойства анимации:

  • animation-name: имя анимации, указанное в @keyframes;
  • animation-duration: продолжительность анимации;
  • animation-timing-function: функция распределения (например, ease, linear);
  • animation-delay: задержка перед запуском анимации;
  • animation-iteration-count: количество повторов (или infinite для бесконечного цикла);
  • animation-direction: направление (например, normal или reverse).

Пример: Вращающийся логотип.

@keyframes rotate {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}

.logo {
  width: 100px;
  height: 100px;
  background-color: #f8c291;
  border-radius: 10%;  /* для примера можно попробовать разные значения */
  animation: rotate 3s linear infinite;
}

3. Трансформации: Превращаем и перемещаем элементы

Что такое трансформации (transform)?

Трансформации позволяют изменять форму, размер, положение и наклон элементов. С их помощью можно создавать эффекты масштабирования, вращения, смещения и наклона.

Основные функции:

  • translate(): перемещение по осям X и Y;
  • rotate(): вращение элемента;
  • scale(): масштабирование;
  • skew(): наклон.

Пример: Элемент, который при наведении увеличивается и поворачивается.

.transform-box {
  background-color: #62e326;
  width: 150px;
  height: 150px;
  transition: transform 0.5s ease;
}

.transform-box:hover {
  transform: scale(1.2) rotate(15deg);
}

4. Расширенная информация и практика

Дополнительные свойства анимаций

Помимо базовых свойств, можно использовать:

  • animation-fill-mode: Определяет, как стили анимации применяются до и после её выполнения (например, forwards сохранит конечное состояние);
  • animation-play-state: Позволяет останавливать и запускать анимацию (paused или running);
  • animation-direction: Изменяет направление анимации, позволяя ей играть в обратном порядке (reverse или alternate для чередования направлений).

Переходы и анимации можно комбинировать для создания сложных эффектов. Например, можно задать плавный переход для изменения трансформации, а затем запустить анимацию для непрерывного движения.

Практическое задание: Анимированная кнопка и вращающийся логотип

Задание:
1. Создай кнопку, которая плавно меняет цвет и слегка увеличивается при наведении.
2. Создай логотип, который непрерывно вращается.

Пример кода для анимированной кнопки:

button {
  background-color: #7826e3;
  color: white;
  padding: 12px 24px;
  border: none;
  border-radius: 5px;
  transition: background-color 0.5s ease, transform 0.5s ease;
}

button:hover {
  background-color: #f8c291;
  transform: scale(1.1);
}

Пример кода для вращающегося логотипа:

@keyframes spin {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}

.logo {
  width: 120px;
  height: 120px;
  background-color: #f8c291;
  border-radius: 50%;
  animation: spin 4s linear infinite;
}
