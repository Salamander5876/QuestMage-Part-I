Библиотеки JavaScript для веб-разработки

В этом свитке собраны самые могущественные заклинания (библиотеки) для твоих веб-проектов. Освой их, и станешь настоящим архимагом интерфейсов!

Как подключать JavaScript к HTML

Прежде чем использовать сторонние библиотеки, нужно знать, как подключать JavaScript к HTML. Это основа для работы с любым кодом.

Используй тег <script> с атрибутом src, чтобы подключить внешний файл:

<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Подключение JS</title>
</head>
<body>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.11.3/gsap.min.js"></script>
</body>
</html>

Объяснение:

  • Тег <script> размещается в <body> (обычно в конце).
  • Атрибут src указывает путь к файлу — локальному или через CDN.
  • CDN (Content Delivery Network) — это быстрый способ подключить популярные библиотеки.

Библиотеки для анимации

GSAP (GreenSock Animation Platform)

Мощная библиотека для создания плавных и производительных анимаций. Работает с CSS, SVG, Canvas и другими элементами.

Пример кода:

<script src="https://cdnjs.cloudflare.com/ajax/libs/gsap/3.11.3/gsap.min.js"></script>
<div id="box" style="width:100px;height:100px;background:red;"></div>
<script>
  gsap.to("#box", { duration: 2, x: 300, rotation: 360 });
</script>

Ссылка: Официальный сайт GSAP

Примеры работ: GSAP Showcase

Anime.js

Лёгкая библиотека с простым API для анимации CSS, SVG, DOM-атрибутов и JavaScript-объектов.

Пример кода:

<script src="https://cdnjs.cloudflare.com/ajax/libs/animejs/3.2.1/anime.min.js"></script>
<div id="box" style="width:100px;height:100px;background:blue;"></div>
<script>
  anime({
    targets: '#box',
    translateX: 250,
    rotate: '1turn',
    backgroundColor: '#FFF',
    duration: 800
  });
</script>

Библиотеки для математики

Math.js

Расширенная библиотека для математических вычислений. Поддерживает числа, матрицы, единицы измерения и т.д.

Пример кода:

<script src="https://cdnjs.cloudflare.com/ajax/libs/mathjs/10.0.0/math.min.js"></script>
<script>
  const result = math.evaluate('sqrt(16) + sin(pi / 2)');
  console.log(result); // 5
</script>

Ссылка: Официальный сайт Math.js

D3.js

Библиотека для интерактивных визуализаций данных с использованием SVG, HTML5 и CSS.

Пример кода:

<script src="https://d3js.org/d3.v7.min.js"></script>
<svg width="100" height="100"></svg>
<script>
  d3.select("svg").append("circle").attr("cx", 50).attr("cy", 50).attr("r", 40).style("fill", "purple");
</script>

Ссылка: Официальный сайт D3.js

Библиотеки для 3D

Three.js

Популярная библиотека для создания 3D-графики с WebGL. Поддерживает сцены, камеры, свет и тени.

Пример кода:

<script src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r128/three.min.js"></script>
<div id="container"></div>
<script>
  const scene = new THREE.Scene();
  const camera = new THREE.PerspectiveCamera(75, window.innerWidth / window.innerHeight, 0.1, 1000);
  const renderer = new THREE.WebGLRenderer();
  renderer.setSize(window.innerWidth, window.innerHeight);
  document.getElementById('container').appendChild(renderer.domElement);
  const geometry = new THREE.BoxGeometry();
  const material = new THREE.MeshBasicMaterial({ color: 0x00ff00 });
  const cube = new THREE.Mesh(geometry, material);
  scene.add(cube);
  camera.position.z = 5;
  function animate() {
    requestAnimationFrame(animate);
    cube.rotation.x += 0.01;
    cube.rotation.y += 0.01;
    renderer.render(scene, camera);
  }
  animate();
</script>

Ссылка: Официальный сайт Three.js

Примеры работ: Примеры Three.js

Babylon.js

Мощный 3D-движок для игр и приложений в браузере. Поддерживает физику, анимацию и звук.

Пример кода:

<script src="https://cdn.babylonjs.com/babylon.js"></script>
<canvas id="renderCanvas"></canvas>
<script>
  const canvas = document.getElementById("renderCanvas");
  const engine = new BABYLON.Engine(canvas, true);
  const scene = new BABYLON.Scene(engine);
  const camera = new BABYLON.FreeCamera("camera", new BABYLON.Vector3(0, 0, -10), scene);
  const light = new BABYLON.HemisphericLight("light", new BABYLON.Vector3(0, 1, 0), scene);
  const box = BABYLON.MeshBuilder.CreateBox("box", { size: 2 }, scene);
  engine.runRenderLoop(() => {
    scene.render();
  });
</script>

Другие распространённые библиотеки

jQuery

Классическая библиотека для упрощения работы с DOM, событиями, AJAX и анимациями.

Пример кода:

<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
<button id="btn">Нажми меня</button>
<script>
  $("#btn").click(function() {
    alert("Привет, мир!");
  });
</script>

Ссылка: Официальный сайт jQuery

Lodash

Утилитарная библиотека с функциями для работы с массивами, объектами, строками и т.д.

Пример кода:

<script src="https://cdn.jsdelivr.net/npm/lodash@4.17.21/lodash.min.js"></script>
<script>
  const arr = [1, 2, 3, 4];
  const sum = _.sum(arr);
  console.log(sum); // 10
</script>
